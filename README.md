# BigData
>> 大数运算：大数运算，顾名思义，就是对很大的数进行一系列的运算。在数学中，数值的大小是没有上限的，但是在计算机中，由于字长的限制，计算机所能表示的范围有限，对于很大的数，计算机无法对其进行直接计算，需要用到所谓的高精度算法，即用数组来存储整数，并模拟手算的方法进行四则运算。
# BitMap
